Inspiration

Growing up, video games have piqued my curiosity especially regarding how it was made. With various complex mechanics and graphics, it left a significant impact on my young mind. With this in mind, I decided to pursue a career in an industry that explains the mechanics and theory behind gaming and how it all comes together.

One of my dream goals is to be able to create an application whether on mobile or desktop. As I continue to progress in my degree, I've grown to love creating applications as it allows you to analyze and understand both parties; the user and the developer. Coding has brought so many new perspectives into my life, some of which have garnered positive experiences and ideals.

Coding Projects

The mission of “Pin-It” enterprise is to develop intuitive tools for users to keep track of all their diverse tasks to help achieve a balanced lifestyle. Our goal is to make the application the most accessible as possible and easy to use. The design catered to young adults but the functionality for everyone. There’s a need for a user friendly task manager that allows for a nice visual display of the daily tasks while also allowing weekly planning. Our website fulfills this need by offering a daily view of your tasks with color-coded categories, a pin option for urgency and option to keep track of completed tasks. This application was done in a group of 3 while performing Agile and Scrum principles. This project was created by Emmanuelle Oleszczuk, Sarim Syed and Angela Shanmugam.
Project - Stick It

The purpose of the homebudget program is to create a manageable tool to help users organize their expenses. Initial base code was taken from Helen Katalifos and Sandy Bultena. Inventory Tracker is a project which includes saving and loading data from files. This Inventory Tracker will allow a grocery owner to track their items. The app will be able to let the owner add and remove products. They will also be allowed to update their available quantities and suppliers. Generated reports about the items are also included. This project was created by Angela Shanmugam and Reigina Mae Martin.
